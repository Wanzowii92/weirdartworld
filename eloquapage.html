  • 용어 정리
    *Contact : 엘로콰에 등록되는 고객을 의미
    *Shared list : 특정 캠페인을 진행했거나 특정 캠페인 진행을 원하는 Contact을 모아둔 리스트 (Form processing 과정에서 세팅 가능)
  • SEGMENT
    1. 정의
      2. *Contact을 원하는 조건에 맞춰서 List화하기 위함
    2. 상세 기능
      3. *조건 필터 걸어서 전체 Contact 필터링 후, 원하는 contact만 List화 (각 Form/Email/campaign 참여한 contact 데이터도 활용 가능)

      *Shared list는 캠페인에 사용하기 위해서 무조건 Segment화 필요 (Make local copy는 현재 데이터 그대로 / Add shared list는 업데이트 되면 같이 업데이트)

      *Exclude / Include를 통해 contact 필터링 가능

      *Merge - 합집합 / Intersect - 교집합 / Trim - 차집합 (필터 걸린 데이터들끼리 계산 가능)

      *특정 필터 shared 기능으로 저장 가능 (이후 같은 필터로 Segment 설정 가능)
